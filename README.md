# Product Store App

A modern product store application built with React and Redux Toolkit.

## Features

- Product cards
- Shopping cart
- Add to cart
- Remove from cart
- Total price calculation
- Responsive layout
- Modern UI design

## Technologies Used

- React
- Redux Toolkit
- Vite
- JavaScript

```bash
npm install
npm run dev
